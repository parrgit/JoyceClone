Projects

Photobox

The Friday/PA team helped the Photobox in-house dev team to move their old website onto a new stack using React and GraphQL.

As part of my role I helped set up a grid system using CSS Grid. We used mixins so that it could easily be used be other teams across the business. I also helped set up font rules and branding in a way that it could easily be replaced with branding of the sister companies of the Photobox Group.

As we were working on a live site as part of the client's development team we continuously deployed features as soon as they were built and tested.

Photobox Editor

Photobox's old photo editor was still running on Flash and with many browsers no longer supporting Flash it was time to update it to a more modern stack with React. I joined the team during the last two months before handover so I had to quickly get up to speed with a complex code base. I used my expertise from the main Photobox website to set up branding for the editor so that it could also be used for their sister companies. I also implemented a couple of new features like swapping images.

Nuffield Health gym joining

In 2016 we first built Nuffield Health's online gym joining platform. Prior to this you couldn't join a Nuffield gym online. Gym joining has since had a few interations, all of which I was a part of. The site is built on a Ruby on Rails backend. The frontend is using a more traditional setup using HTML, scss and JavaScript.
